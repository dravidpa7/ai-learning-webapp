Activation Function Neural Network

Sigmoid (logistic)

Folmula:

\( P(y = 1 \mid X) = \text{sigmoid}(z) = \frac{1}{1 + e^{-z}} \)

Range: (0, 1) Use case: Probability problems

Real-time example:

  • Spam detection: output tells how likely an email is spam (e.g., 0.9 → 90% spam).
  • Logistic regression output neuron.

Tanh (Hyperbolic Tangent)

Folmula:

\( \tanh(z) = \frac {e^z - e{-z}}{e^z + e{-z}}\)

Range:(-1, 1) Use case:When you need zero-centered output.

Real-time example:

  • Sentiment score (e.g., product review scoring: -1 → negative, +1 → positive).
  • RNN hidden layers (better convergence than sigmoid).

ReLU (Rectified Linear Unit)

Folmula:

\( \text{ReLU}(z) = \max(0, z) \)

Range:[0, ∞] Use case:Deep neural networks — default for most modern networks.

Real-time example:

  • Image recognition (CNNs): activates only when features are detected (e.g., edge present → positive activation, no edge → 0).
