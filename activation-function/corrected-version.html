Formula:

\( P(y = 1 \mid X) = \text{sigmoid}(z) = \frac{1}{1 + e^{-z}} \)

Range: (0, 1) | Use case: Probability problems

  • Binary classification in logistic regression

Tanh (Hyperbolic Tangent)

Formula:

\( \tanh(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}} \)

Range: (-1, 1) | Use case: Zero-centered output

Real-world Examples:

  • Sentiment analysis: Score from -1 (negative) to +1 (positive)
  • RNN hidden layers for better gradient flow
  • Feature normalization in preprocessing
